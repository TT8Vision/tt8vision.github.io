# IMADETHEARK — Photo File Guide

Save your Instagram photos here with these exact filenames and they'll
automatically appear on the site. No code changes needed.

## Hero (full-screen background)
| File | What to use |
|------|-------------|
| `hero-bg.jpg` | Any moody Cape Town street/lifestyle shot — the man walking, outdoor shots. Displays very dark (brightness 18%) so any photo works. |

## About Section (4:5 portrait frame, left side)
| File | What to use |
|------|-------------|
| `about-lifestyle.jpg` | Group community shot OR the older man in the black ARK tee OR the couple shot |

## Product Cards (3:4 portrait frame)

## Tips
- Any format works: .jpg .jpeg .png .webp
- Recommended size: at least 800px wide for product cards, 1400px+ for hero
- The site handles zoom/crop automatically via object-fit: cover
- Placeholders show automatically if a file is missing — nothing breaks
